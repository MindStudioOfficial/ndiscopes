# Contributing to NDIScopes
We ♥ your input! We encourage and value any form of contributions:
- Reporting a bug
- Discussing the current state of the code
- Submitting a fix
- Proposing new features
- Becoming a maintainer
- Writing documentation

## I have a question
If you have a question make sure that you have looked into open and closed [Issues](https://github.com/MindStudioOfficial/ndiscopes/issues?q=) of our repo. 
If you still think that your question is unanswered feel free to open a [new Issue](https://github.com/MindStudioOfficial/ndiscopes/issues/new) and give it the 
[🟣question] tag. We will try our best to answer as soon as possible and hopefully clear up any uncertainties.

## I want to report a bug 🐞

Reporting any bugs helps us a lot because some of them are really hard to find and it encourages us to fix it as quickly as possible.
Before reporting anything make sure you are using the latest release. 
If the bug persists please create a [new Issue](https://github.com/MindStudioOfficial/ndiscopes/issues/new) and give it the 
[🔴bug] tag.

In the issue make sure that you provide the following information:
- what you would expect to happen vs. the actual behavior
- steps to reproduce whyt you have observed
- screenshots or better a formatted copy of any error messages that might appear
- if it is a visual bug screenshots are very helpful
- if you know where the bug originates from in the code make sure that you format the code using github markdown and provide only relevant parts

## I want to suggest an Enhancement 📈

Before requesting a new feature make sure that it is not yet present in the [Roadmap](https://github.com/MindStudioOfficial/ndiscopes#roadmap) of the project.
Also please check that there aren't any [Issues](https://github.com/MindStudioOfficial/ndiscopes/issues?q=) regarding a similar feature request.
After that feel free to create a [new Issue](https://github.com/MindStudioOfficial/ndiscopes/issues/new) and give it the 
[ⓂFeature Request] tag.

## I want to add to the documentation 📖

If you feel like the documentation is missing something or has wrong information please create a [new Issue](https://github.com/MindStudioOfficial/ndiscopes/issues/new) and give it the 
[🔵documentation] tag.
